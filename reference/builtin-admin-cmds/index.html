内置管理命令（配置参考）

约 309 字大约 1 分钟

2026-03-23

PluginConfig 中与内置 ! 命令相关的字段；行为说明见 插件开发 · 内置管理命令。

源码：ncatbot/plugin/builtin/system_manager/main.py

配置字段

字段类型默认说明
enable_builtin_commandsbooltrue是否启用内置 ! 管理命令（仍要求发送者为 root）。
builtin_commandsBuiltinCommandsConfig全 true子命令开关：reload / load / unload / sysinfo / config_toggle / plugins / indexed / commands / filter。
builtin_commands_group_allowedboolfalse是否允许在群聊中使用（默认仅私聊）。

查询命令

字段类型默认说明
pluginsbooltrue启用 !plugins — 列出已加载插件
indexedbooltrue启用 !indexed — 列出已索引但未加载的插件
commandsbooltrue启用 !commands — 列出已注册的命令
filterbooltrue启用 !filter — 管理分发过滤规则

分发过滤命令

!filter 命令需要 dispatch_filter 服务可用。

子命令用法说明
!filter add!filter add <group|user> <id> <plugin> [cmd1,cmd2]添加过滤规则
!filter remove!filter remove <rule_id>移除指定规则
!filter list!filter list [group|user] [id]查询规则
!filter clear!filter clear [plugin]批量清除规则

相关全局键

键说明
root管理员 QQ，与发送者比对后方可执行命令。
